# Installation

## Steps
* `git clone https://github.com/ayushanand18/imdb-movie-service`
* `cd imdb-movie-service`

## Frontend
* `cd next-frontend`
* `npm install`

## Backend
* `pip install -r requirements`
    Or if you use a `venv`, first do `source .venv/bin/activate` and then, `pip install -r requirements.txt`
